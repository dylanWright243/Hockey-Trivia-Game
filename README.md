# Hockey Trivia Game

A multiple-choice hockey trivia game built with Processing.

## Features
- 15 hockey trivia questions
- Interactive multiple-choice interface
- Score tracking
- Play again option

## How to Run
1. Install [Processing](https://processing.org/download/)
2. Clone this repository
3. Open `HockeyTrivia.pde` in Processing
4. Click the run button

## Code Structure
- `setup()`: Initializes questions and answers
- `draw()`: Handles all screen rendering
- `mouseClicked()`: Processes user interactions
